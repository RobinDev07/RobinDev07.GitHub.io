LLCMDMenu V3.0.2

配置文件


/config/config.json 通用配置
  - hub 主城坐标
    - type
    - value
  - langCode 语言
  - triggerItem 用于打开菜单的物品
  - showWelcome 是否展示欢迎Toast
  - scores 记分板项
    - $记分板名称
      - info 是否在个人信息页展示
/config/icons.json 图标索引
  - CMDMenu
    - $翻译文本 对应材质路径
/config/index.json 菜单索引
/config/mails.json 邮件配置
