LLCMDShop V1.1.0

配置文件


/config/config.json 通用配置
  - langCode 语言
  - killBonus 击杀奖励
    - $奖励名称
      - type
      - value
      - max
      - min
    - $生物标准类名 对应奖励名称
    - scores 记分板项
      - $记分板名称
        - transfer 是否用于自助转账
/config/icons.json 图标索引
  - CMDShop
    - $翻译文本 对应材质路径
/config/shop.json 商品索引
  - index 商品主索引
  - $商品索引
  - giftCode 礼品码
